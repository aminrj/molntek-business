# Molntek Business

Business operations, strategy, and deliverables for Molntek — the independent AI agent security consultancy.

## Structure

```
strategy/          ← Consulting offers, GTM, pricing, positioning
artifacts/         ← One-pagers, checklists, lead magnets (PDFs)
case-studies/      ← HolmesGPT, client writeups
presentations/     ← Slidev decks, talk materials
operations/        ← Tracker, CFP tracker, NEXT.md, DASHBOARD.md
website/           ← molntek-astro-website code
```

## Key Files

- **`operations/NEXT.md`** — The one file to open every morning. What to work on today.
- **`operations/DASHBOARD.md`** — Weekly review: progress, metrics, CFP pipeline.

## Positioning

> "I help teams running production AI agents fix the thing their scanners can't find: whether they can actually stop a misbehaving agent."
